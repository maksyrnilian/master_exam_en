% full log lives in build/<rootfile>.log (matches your -outdir setting)
% search log for "! " (exclamation + space) to find fatal errors fast
% search log for "Undefined" for missing refs, citations, or control sequences
% error format: "l.47" means the problem is on line 47 of your source
% last lines of log confirm success ("Output written on main.pdf ...")
%   or failure ("Fatal error occurred, no output PDF file produced")
% log file location: build/<rootfile>.log (matches your -outdir setting)

%   "Emergency stop... no legal \end found"
